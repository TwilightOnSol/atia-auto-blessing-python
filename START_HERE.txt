🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

   Python not found?
   → Install Python and check "Add Python to PATH"
   → Restart computer
   → Run install.bat again

   Permission denied?
   → Right-click install.bat → "Run as administrator"

   Installation failed?
   → Check internet connection
   → Run as administrator
   → Try again

   Not sure about setup?
   → Run check.bat to diagnose
